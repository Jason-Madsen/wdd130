This assignment was to make a page to showcase all of the living Apostles. It taught me about grid positioning and using more CSS effects.

This assignment was to create a Halloween themed page without changing any HTML, only CSS, it taught me that you can have fun designing and you can do so much without changing the HTML.

This assignment was to create a page for our favorite devionals, it taught me how I can better design and make forms and tables.

CSS Positioning

This assignment was to recreate shapes and colors with only CSS and grids. It showed me how to place things over each other and overlap them within a grid.

Gallery

In this assignment I had to take the images and make them smaller so that the page would load quicker. I learned that images don't have to be so big to have good quality.
